# Customer Cohort Analysis & Market Basket Intelligence

An end-to-end data analytics pipeline that combines customer segmentation and association rule mining on 800K+ retail transactions, with results visualized in a Power BI dashboard.

---

## Project Overview

This project analyzes the [Online Retail Dataset](https://www.kaggle.com/datasets/lakshmi25npathi/online-retail-dataset) (Kaggle) to identify distinct customer cohorts and uncover cross-sell opportunities across the top 5 countries by transaction volume.

**Techniques used:** FP-Growth, Apriori, K-Means Clustering, Agglomerative Clustering, RFM Segmentation, PCA

---

## Key Findings

- **805,549** valid transactions after cleaning (from 1,067,371 raw records)
- Netherlands yields the highest association rule lift (avg **29.3**), indicating the strongest product affinities
- FP-Growth and Apriori produce identical rule sets — FP-Growth preferred for runtime efficiency
- UK customers segment cleanly into **2 cohorts** (silhouette = 0.94); Germany and France best at K=3
- Cluster-specific rules reveal distinct product preferences per segment, enabling targeted recommendations
- RFM segmentation classifies customers into Champions, Loyal, At Risk, and Lost for retention strategy

## Methods

### Data Cleaning
- Removed cancelled invoices, null CustomerIDs, zero-quantity and zero-price records
- Engineered `TotalPrice = Quantity × UnitPrice`

### Association Rule Mining
- Built country-level basket matrices per invoice
- Applied **FP-Growth** and **Apriori** (min support = 0.03, min lift = 1.0)
- Compared both methods across avg lift, confidence, and support

### Customer Clustering
- Features: TotalSpending, NumTransactions, NumProducts, TotalQuantity, AvgOrderValue
- StandardScaler normalization
- Best K selected via silhouette score (K = 2–5 range)
- Both **K-Means** and **Agglomerative Clustering** applied and compared

### RFM Segmentation
- Recency computed from last purchase date
- Customers labeled: Champion, Loyal, At Risk, Lost

### Cluster-Specific Rules
- FP-Growth re-run per cluster within each country
- Surfaces product recommendations tailored to each customer segment
